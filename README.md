#  RaktSetu — Connecting Lives, One Drop at a Time

## Overview

RaktSetu is a smart blood donation platform designed to connect donors, requesters, and blood banks efficiently during emergencies.
It focuses on fast decision-making through intelligent matching and a clean user experience.



## Problem Statement

Finding blood donors during emergencies is often slow and unreliable.
Many existing systems lack proper matching, clarity, and backup options.

---

## Our Solution

RaktSetu provides:

* Smart donor matching based on user input
* Clear visibility of best available donors
* Nearby blood banks as backup options
* A simple and fast interface for urgent situations


##Key Features

### Smart Matching System

* Match score based on location & activity
* Highlights best donor
* Explains why a donor is selected

### Emergency Mode

* Prioritizes urgent requests
* Highlights most suitable donors

### Blood Bank Integration

* Displays nearby blood banks along with donors
* Provides fallback when donors are limited

### Donor Insights

* Last donation time
* Availability indication
* Verified donor indicator

### Clean UI/UX

* Fast and intuitive interface
* Designed for emergency use

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Data Handling:** Static / Mock Data
* **Version Control:** Git & GitHub

---

##  How It Works

1. User enters request details
2. System filters and matches donors
3. Displays:

   * Best matching donors with score
   * Nearby blood banks
4. User can directly contact donors
